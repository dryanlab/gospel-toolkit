# 真理磐石 版权证据存档

本目录保存颜弟兄（Su Yan）深度参与内容创作的证据记录。

## 目的
证明真理磐石的原创内容是在人类创作者的创造性指导、审阅和修改下完成的，
而非纯AI自动生成。

## 记录内容
每条记录包含：
- **日期和时间**
- **内容标识**（哪篇文章/哪个章节）
- **参与类型**：构思、选题、审阅、修改指令、神学判断、内容决策、发布决策
- **具体内容**：颜弟兄的原始指令或修改意见（引用原文）
- **结果**：修改前后的对比

## 文件命名
- `YYYY-MM.md` — 按月存档

## 补充证据
- Git commit 历史（`git log`）
- Telegram 对话记录
- REVIEW-CHECKLIST.md 审阅记录
